--- troubleshoot.io ---

Purpose:
- Allows users to be able to research and find common problems to technical issues related to hardware, network, and software.

Structure of the Page: 
- Home Directory
    - Consists of navigation bar, common issues of the week, and user testimonials, and footer leading to other links.
- Navigation Bar: Home, Categories, About, Contact, Explore (defective)
- Categories:
    - Hardware Issues
      - RAM Error
    - Network Issues
      - Router Issue
    - Software Issues
      - Excel Crash
- About: Purpose of the web site and intended audience
- Contact: Leads to contact form where user can input information
    - After submission, leads to submit.html where there is a thank you message with a video and sound that a user can manually play.
